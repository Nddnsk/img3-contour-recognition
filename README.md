Контурное распознавание объектов (алгоритм Протасова)

Репозиторий содержит реализацию алгоритма распознавания объектов по их контуру,
основанного на методе В. И. Протасова
«Распознавание трёхмерных объектов на сложном фоне по части контура» (2001).

Алгоритм предназначен для распознавания объектов по полному или частичному контуру
и инвариантен к повороту, масштабу и сдвигу.

----------------------------------------------------------------

Краткое описание алгоритма

1. Контур объекта задаётся упорядоченным набором точек.
2. Контур аппроксимируется ломаной линией с помощью поиска характерных точек
   (рекурсивное разбиение с параметром EPS).
3. Для каждого отрезка между характерными точками вычисляется набор признаков:
   N1, N2 — отношения длины текущего отрезка к длинам сторон прилегающих
   треугольников, не являющихся отрезками контура;
   Z — признак ориентации, определяющий, находятся ли соседние отрезки
   по одну или по разные стороны от текущего.
4. Контуры сравниваются как последовательности признаков с учётом циклического
   сдвига и допуска частичного совпадения.

----------------------------------------------------------------

Структура проекта

src/        — реализация алгоритма
data/       — эталонные и тестовые контуры
CMakeLists.txt
README.md

----------------------------------------------------------------

Формат файлов контуров

N
x1 y1
x2 y2
...
xN yN

----------------------------------------------------------------

Тестовые данные

В проекте используются контуры следующих объектов:
самолёт,
танк,
корабль.

Для каждого объекта сгенерированы различные ракурсы и масштабы,
а также частичные и неполные контуры.

----------------------------------------------------------------

Сборка проекта

mkdir build
cd build
cmake ..
make

----------------------------------------------------------------

Результаты

Подтверждена инвариантность алгоритма к масштабу и повороту.
Объекты разных классов корректно различаются.
